
# Landing Astro

## Description
A simple landing page project built with Astro.

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Project Structure
- `/src` - Source files
- `/public` - Static assets
- `astro.config.mjs` - Astro configuration
